# Plataforma Educativa SMTP - Semana 4

Esta plataforma ha sido desarrollada para el estudio técnico del protocolo SMTP (Simple Mail Transfer Protocol), correspondiente a la **Semana 4** del curso de Arquitectura de Redes.

## Características
- **Interfaz Moderna**: Dashboard estilo ingeniería con diseño responsivo.
- **Sección Antes**: Preparación con video introductorio.
- **Sección Durante**: Acceso a documentos de teoría y guías de laboratorio (Postfix en Linux).
- **Simulador Interactivo**: Terminal web que simula una sesión SMTP real (HELO, MAIL FROM, RCPT TO, DATA, QUIT).

## Requerimientos de la Semana
- **Teoría**: MTEO-S04-LDIAZ-2025-02
- **Laboratorio**: PLAB-S04-LDIAZ-2026-01 (Entrega: 16 de abr, 20 pts)

## Cómo usar
Simplemente abre el archivo `index.html` en cualquier navegador moderno.
